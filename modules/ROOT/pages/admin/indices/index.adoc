要想访问indices JavaAPI，你需要从 xref:admin/index.adoc#java-admin[AdminClient] 类中调用 `indices()` 方法:

[source,java]
--------------------------------------------------
IndicesAdminClient indicesAdminClient = client.admin().indices();
--------------------------------------------------

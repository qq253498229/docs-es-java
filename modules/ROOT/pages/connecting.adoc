[[connecting]]
= 连接

{java-client} 中包含三大重要组件：

* **客户端API类**。 一些为 {es} API提供强类型数据结构和方法。
由于 {es} API 非常庞大，所以它会按照功能来进行分组（也可以叫 “命名空间“），每个分组都会有它们特有的客户端类。
{es} 核心功能会在 `ElasticsearchClient` 类中实现。
* **JSON对象映射器**。 它会将你的应用业务类与一个JSON对象进行映射，这样就能无缝的与 API 客户端进行无缝集成了。
* **传输层实现**。 这是负责处理所有 HTTP 请求的地方。

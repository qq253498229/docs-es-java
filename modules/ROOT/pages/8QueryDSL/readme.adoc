[#_领域查询]
= 领域查询

ES以类似REST https://www.elastic.co/guide/en/elasticsearch/reference/5.6/query-dsl.html[Query DSL] ( https://www.elastic.co/guide/cn/elasticsearch/guide/cn/query-dsl-intro.html[中文] )的方式提供了完整的表达式查询。

可以通过 **QueryBuilders** 构建查询。构建成功后就可以使用 xref:6SearchAPI/readme.adoc[Search API]。

使用 **QueryBuilders** 之前需要在类中引入 ：

[source,java]
----
import static org.elasticsearch.index.query.QueryBuilders.*;
----

注意你可以使用 **QueryBuilder** 类中的 **toString()** 方法轻松的打印(调试)JSON生成的查询语句。

**QueryBuilder** 可以被用于接受查询的任何接口中，例如 **count** 和 **search** 。
